markdown格式

发表于 2021-02-02 更新于 2021-02-04 分类于 教程 ， 博客 ， 学习 ， 编程 阅读次数：

标题：

注意：#与标题之间要有空格才能被识别解析

分别用1-6个#表示不同级别的标题

# h1级标题
## h2级标题
### h3级标题
#### h4级标题
##### h5级标题
###### h6级标题

分割线：

三个以上的短线--- 即可作出分割线,效果如下

超链接：

[连接名称](网址 , 标题)
注意必须是英文括号。

我是链接名
点我刷新

[蔡正义](https://www.caizhengyi.com)

另一种超链接写法：[链接名][链接代号]
[here][3]

然后在别的地方定义 3 这个详细链接信息，
[3]: http://www.izhangbo.cn “聚牛团队”

直接展示链接的写法：<http://www.izhangbo.cn>

换行缩进形成代码区块

这里先换行，然后缩进4个空格，之后的内容便可以原样显示了，适合用于显示代码内容。直到文本结束或最后一个存在缩进的行为止。    
void main()
{
printf("Hello, Markdown.");
}

块引用

给引用的文本开始位置都加一个 ‘>’，
便可组成一个块引用。在块引用中，可以结合
其他markdown元素一块使用，比如列表。也可以只在第一行加大于号，其他位置不加。

  • 块引用里使用列表，需要和上面的内容隔开一个空行
  • 记得加空格哦。
